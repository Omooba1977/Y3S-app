# Y3S — Say YES to Financial Freedom
## Frontend PWA

Africa's AI-powered financial freedom coaching app.

### Tech Stack
- Single-file PWA (HTML + CSS + JS)
- Hosted on Netlify
- Backend: Railway (Node.js)
- AI: Anthropic Claude Haiku

### Files
- `index.html` — The complete app (397KB)
- `sw.js` — Service worker for offline support and caching
- `manifest.json` — PWA manifest for install prompt

### Deployment
Connected to Netlify for automatic deployment on every commit.
